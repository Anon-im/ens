# Linea ENS POH

A NestJS API responsible for returning a signature aknowledging an address has passed the POH process.  
Uses the [POH api](https://linea-xp-poh-api.linea.build) to check if an address has a POH.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
